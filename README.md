# 2026.4.5 实验二

## 实验内容

### （1）UML 类图绘制
按照规范绘制简单工厂模式与工厂方法模式的结构类图。

### （2）设计模式应用
完成代码重构，使用简单工厂模式实现道具创建，使用工厂方法模式实现敌机创建。

### （3）敌机生成
界面上可观察到 4 种敌机（Boss 敌机本次迭代不要求出场）按周期随机出现。

### （4）敌机移动
精锐和王牌敌机向屏幕下方移动，且可左右移动。

### （5）敌机攻击
- **精锐敌机**：按设定周期向下直射双排子弹
- **王牌敌机**：采用扇形散射弹道，单次同时发射 3 颗子弹

### （6）道具生成
精锐敌机、王牌敌机被击毁时按概率随机掉落某一种道具：
- **精锐敌机**：可掉落 4 种道具（不含冰冻）
- **王牌敌机**：可掉落全部 5 种道具

### （7）道具效果
除加血道具外，其余道具生效时在控制台输出提示信息即可。

# 2026.4.16 实验三

## 实验内容

### （1）UML 类图绘制

按照规范绘制策略模式的 UML 结构类图；

### （2）设计模式应用

完成代码重构，采用策略模式实现不同机型的弹道发射逻辑，以及两种火力道具的加成效果；

### （3）敌机生成

游戏界面可正常显示 5 种敌机，按预设周期随机出现；当玩家分数达到指定阈值时，触发 Boss 敌机生成；

### （4）敌机移动

Boss 敌机悬浮于界面上方，仅执行左右移动操作，不向下移动；

### （5）敌机攻击

Boss 敌机采用环射弹道，单次同时发射 20 颗子弹；

### （6）道具生成

Boss 敌机被击毁时，随机掉落 3 个道具（道具类型无限制）；

### （7）道具效果

火力道具生效后，英雄机将当前弹道切换为散射弹道；超级火力道具生效后，英雄机将当前弹道切换为环射弹道；炸弹、冰冻道具生效时，在控制台输出对应提示信息即可。
